### 구체적인 실행 절차

#### 1단계: 데이터 수집 시스템 구축
```
✅ 이미 준비됨:
- NewsAPI (이미 키 보유)
- 텔레그램 연결 완료
- 주간 일정 cron job 설정됨

📝 실제 코드:
1. 뉴스 수집
   - Country: kr (한국)
   - Category: business (경제)
   - PageSize: 100 (더 많은 뉴스)

2. 필터링
   - 최신 순 정렬
   - 관련성 높은 뉴스 필터링
   - 중복 제거

3. 데이터 저장
   - 디렉토리: ~/openclaw/data/economy-news/
   - 파일명: YYYY-MM-DD.json
   - 자동화: 매일 자동 생성
```

#### 2단계: 데이터 분석 로직 구축
```
🔍 키워드 분석:
1. 뉴스 제목에서 키워드 추출
   - 예: "경제", "금리", "반도체", "삼성전자"

2. 빈도수 계산
   - 각 키워드의 등장 횟수
   - 상위 10개 키워드 선정

3. 트렌드 분석
   - 이전 주와 비교
   - 증가/감소 추적
```

#### 3단계: 보고서 생성 및 전송
```
📄 보고서 구조:
1. 핵심 뉴스 5건 (요약)
2. 키워드 분석 결과
3. 시장 반응 데이터
4. 다음 주 전망

📤 전송:
- 텔레그램: 자동 전송 (완료)
- 이메일: PDF 형식으로 보내기 (개선 필요)
```

### 실제 실행 예시

**매주 금요일 오후 5시 실행:**

```python
# python 코드 예시
import requests
import json
from datetime import datetime

# 뉴스 수집
url = "https://newsapi.org/v2/top-headlines"
params = {
    "apiKey": "36eea55a0bbf4b4b8dfff98af3b0a3f8",
    "country": "kr",
    "category": "business",
    "pageSize": 100
}
response = requests.get(url, params=params)
data = response.json()

# 키워드 분석
keywords = {}
for article in data['articles']:
    title = article['title']
    # 키워드 추출 로직...
    for keyword in ['경제', '금리', '반도체', '삼성', '한국경제', ...]:
        if keyword in title:
            keywords[keyword] = keywords.get(keyword, 0) + 1

# 상위 10개 키워드
top_keywords = sorted(keywords.items(), key=lambda x: x[1], reverse=True)[:10]

# PDF 보고서 생성
pdf_report = f"""
📊 주간 경제 뉴스 분석 ({today})
========================

1. 핵심 뉴스 5건
{top_news}

2. 키워드 분석
{top_keywords}

3. 시장 반응
{market_response}

4. 다음 주 전망
{forecast}
"""

# 텔레그램으로 전송
send_to_telegram(pdf_report)
```
